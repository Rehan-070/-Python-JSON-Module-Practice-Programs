# 📦 Python JSON Module Practice Programs

This folder contains Python programs demonstrating how to work with the built-in `json` module for handling JSON data.

Each file focuses on a specific function like loading, dumping, and converting between Python objects and JSON format.

---

## 🚀 Topics Covered

- 📤 dump.py – Convert Python object → JSON file
- 📥 load.py – Read JSON file → Python object
- 🔄 dumps.py – Convert Python object → JSON string
- 🔁 loads.py – Convert JSON string → Python object
- 📄 data.json – Sample JSON data file

---

## 🛠️ Requirements

Make sure you have **Python 3.x** installed.

No external libraries required.

`json` module is built-in.

---

## ▶️ How to Run

# run any specific file  
`python dump.py`

Example:

`python load.py`

Run each file individually to understand its functionality.

---

## 🧪 Example Usage

- Run `dump.py` → Save Python data into JSON file  
- Run `load.py` → Read JSON file into Python  
- Run `dumps.py` → Convert Python object into JSON string  
- Run `loads.py` → Convert JSON string into Python object
